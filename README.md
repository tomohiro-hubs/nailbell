# neil salon bell Webサイト

プライベートネイルサロン「neil salon bell」のための1ページ構成ランディングサイトです。落ち着きのある大人向けトーンと、作品集公開へ向けた導線を兼ね備えたデザインを採用しています。ご提供いただいたベルモチーフのロゴ画像をヘッダー／フッターに配置し、奈良県宇陀市の自宅サロンというコンセプトを表現しています。

## 現在実装済みの主な機能
- ヒーローセクションを中心としたストーリーテリング型LPデザイン
- サロン紹介、特徴紹介、料金メニュー（アクリルスカルプ／ジェルネイル ワンカラー）、ギャラリー、FAQ、カード型のアクセス案内、問い合わせフォーム等の各セクション
- スクロールに追従するグローバルナビゲーションとトップへ戻るボタン
- モバイル対応のナビゲーションメニュー開閉
- FAQアコーディオン（支払いは現金のみ等のサロンポリシーを明記）、問い合わせフォーム・ニュースレターの簡易フロントエンドバリデーション
- レスポンシブレイアウトとアクセシビリティを意識したマークアップ

## エントリーポイント / パス概要
- `/` : トップページ（単一ページ構成）
  - `#hero` / `#about` / `#services` / `#gallery` / `#voice` / `#faq` / `#access` / `#contact` などのセクションアンカー

## 未実装の機能・今後の検討事項
- 作品事例の独立ページおよび詳細ページ
- 実際の予約システム連携、フォーム送信先（現在はデモ動作のみ）
- お知らせ・ブログ等のコンテンツ管理機能
- SNS連携先URL、Googleマップ埋め込みの実住所設定（必要であれば再導入）

## 推奨される次のステップ
1. 作品集専用ページの構築と、indexページからの導線更新
2. 予約フォームを外部予約サービス（例：STORES予約など）と接続、もしくはRESTful Table APIを利用した簡易受付管理の実装
3. 実住所の詳細・営業時間・SNSリンク等の確定情報への差し替え
4. 実際の写真素材への置き換えとギャラリー拡充
5. プライバシーポリシーページの整備とリンク有効化

## プロジェクト概要
- **目的**: neil salon bellの世界観やサービス内容を伝えると共に、予約問い合わせへスムーズに誘導するLPを用意すること
- **主な機能**: サロン紹介（奈良県宇陀市の自宅サロン）、料金・サービス案内（アクリルスカルプ / ジェルネイル ワンカラー）、ギャラリー、FAQ、アクセス案内（Googleマップ非掲載）、問い合わせフォーム
- **デザイン方針**: 大人の女性向けの上品で落ち着いたカラーリング、読みやすさ・回遊性重視のレイアウト
